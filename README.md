# Dustrunner V1

Dustrunner V1 is a small, performance-first arcade rover shooter built for non-gaming computers.

## Design Goal
Create a fast, modern side-scrolling action game focused entirely on gameplay:
- no story screens
- no NPC conversations
- no cutscenes
- instant action
- instant restart
- low-spec friendly

## Core Features
- side-scrolling rover gameplay
- short jump and long jump
- forward fire and upward fire
- hazards, enemies, and checkpoints
- one complete stage
- endless mode
- score system
- lightweight modern visuals
- low-spec mode and basic graphics settings

## Version 1 Scope
Included:
- 1 rover
- 1 biome
- 3 hazards
- 3 enemy types
- 1 stage
- 1 endless mode
- menus, HUD, settings, and score

Excluded:
- story
- dialogue
- bosses
- upgrades
- unlock trees
- multiple biomes
- multiplayer
- advanced destruction

## Technical Direction
- Engine: Godot
- Style: 2.5D fixed-lane arcade action
- Target: office PCs and older laptops
- Priority: performance, clarity, replayability

## Build Philosophy
If a feature does not improve gameplay, clarity, stability, or performance, it does not enter V1.

## Development Order
1. foundation
2. rover movement
3. shooting
4. hazards
5. enemies
6. stage progression
7. UI and score
8. polish
9. performance
10. release
